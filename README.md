# Medical_paper_abstract_classification_using_Tensorflow

## How to run the web app

### To run the client
* ``cd /client/``
* ``npm i ``
* ``npm run dev``

### to run the backend 
* `` cd /server/ ``
* `` pip install Flask Flask-Cors requests tensorflow scikit-learn spacy chardet gunicorn ``
* `` python3 app.py ``
